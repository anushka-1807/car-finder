# 🚗 Car Finder App

A modern React + Material UI web application to browse, search, filter, and wishlist cars — with beautiful UI, animations, and mock API integration.

![Car Finder Screenshot](https://source.unsplash.com/featured/?car,dashboard)

---

## 🌟 Features

- 🔍 **Search cars** by brand or model name
- 🎛️ **Filter cars** by brand, fuel type, seating capacity, and price range
- ❤️ **Wishlist functionality** stored in local storage
- 🎨 **Beautiful UI** with Material UI and Framer Motion animations
- 🔌 **Mock API** integration using [MockAPI.io](https://mockapi.io)
- 📦 Paginated view with 10 cars per page

---

## 📦 Tech Stack

- React (with Hooks)
- Material UI (MUI)
- Framer Motion (for animation)
- JSON Server / MockAPI for mock data
